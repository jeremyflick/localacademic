What The App Enforces

  • Searches 2024+ sources first, then falls back to 2020+ only if needed.
  • Prioritizes stronger evidence signals such as meta-analyses, population-based studies, longitudinal work, and higher-impact sources.
  • Builds 3 to 7 supported items per result when enough relevant abstract evidence is available, including sample sizes and article counts.
  • Uses relevant paper summaries only when no exact statistics are available for that source.
  • Requires source-backed output only.
  • Stays concise and omits unsupported claims.

Known Limits

  • The app relies on source abstracts, not full text.
  • Some papers do not report detailed statistics in the abstract.
  • Citation formatting depends on available metadata from the source record.

How To Get Better Results

  • Use specific search phrases instead of broad topics.
  • Try population + outcome + context, like "adolescents cannabis use depression".
  • Narrow queries when abstracts are too general.
  • Broader queries may return weaker supported items.

Current Live Source Mode

The app uses OpenAlex, Crossref, and Europe PMC for live academic-source discovery, with optional CORE integration available through the backend. It currently relies mainly on abstract text when extracting supported items.
